iOSsampleApp1
=============

- イベント支援サイト「connpass」( http://connpass.com/ )に登録されているイベントから、キーワードで検索したイベントを抽出します。
- connpass が提供する API ( http://connpass.com/about/api/ ) を利用しています。
- json-framework ( http://stig.github.com/json-framework/ )の利用をやめ、NSJSONSerialization を利用するように変更しました。
- 複数キーワードを指定できるようにしました。
- プログラムの基本構成は 沼田哲史「詳解iOS 5プログラミング」（秀和システム ISBN 978-4-7980-3203-0 ）の「Chapter 4 ユーザインタフェースの開発」を参考にしています。
